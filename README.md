## Getting Started

To get a local copy up and running follow these simple example steps.


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/JamesXalis/react-portfolio2.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start your dev server
   ```sh
   npm run dev
   ```
